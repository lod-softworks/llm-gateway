## AI Agents

This directory contains rules, skills, configuration, and other items specific to agentic AI tools.

- **Rules**: 
  - Description: Guidelines, guardrails, and instructions for AI agents working with the repository.
  - Directory (from root): `.agents/rules/`
  - Directory (relative): `rules/`
  - Index:
    - `rules/scope-of-work.md`: Read at task start to stay within requested scope and keep diffs minimal.
    - `rules/dotnet-guidelines.md`: Read before making C#/.NET code, structure, naming, or style decisions.
    - `rules/git-committing.md`: Read before creating commits or writing commit messages.
    - `rules/pull-requests.md`: Read before opening PRs or performing PR/code reviews.
- **Skills**:
  - Description: Agentic AI skills to focus and extend the capabilities of AI agents working with the repository.
  - Directory (from root): `.agents/skills/`
  - Directory (relative): `skills/`
  - Index:
    - No repository-local skills currently exist in `skills/`.
