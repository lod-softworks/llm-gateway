# Scope of Work

When performing tasks, making code changes, or designing a solution, make sure to stay focused and on task.

- Focus on the ask from the user.
- Keep code changes to the minimum required to complete the work.
- Do not make edits, changes, cleanup, or improvements outside of the requested work.
- Suggest additional enhancements or changes to the user.
- Stop and ask clarifying questions about requested work and the scope of changes.
- Git diffs should be easy to read and review.
